    • How to Get Started with Brackets

    (steps to use Brakets )

    1. Make sure you have Brackets downloaded on your device.
    2. Open it, to be sure it works.
    3. Leave Brackets and open files on your device.
    4. Create a folder, name it what you want, ( I am not going to tell you how to live your life)
    5. Make sure the file has the right .________, for instance: .html, .mp4, etc..
    6. Start the document.
    7. First type <!DOCTYPE html>

    8. Then <html> a copy of that should apear next to that or below that, don't erase it, or else you will need to type it in again later. (and that would just be a waste of time)
    9. Then <html>, again do not erase the one that apears with it. Also, make sure the <html> tags are inside the <html> tags.
    10. Then you need a title! for that you should use a <h1> tag, and like all the other tags, another one will apaer, and like all other tags do not erase it.
    11. With the <h1> tag, there are many others, the only difference is that the font size becomes smaller. to do that replace the 1 with 2 or, 3 or, 4 or, 5 or, 6... all that changes is the font size.
    12. Then you may use the <p> tag to add normal text.

    So There You Have It! That, is how you can start to make a website using html
